# Bisection-Calculator
The project is a standalone calculator for computing the bisection of an equation  

## Dependencies  
* Python 3.5 or Python 3.7 as the interpreter
* Virtual Environment (VENV) running in PyCharm  
* Python modules  
  * decimal  
    * Decimal  
 * py_expression_eval    
    * Parser  
 * tkinter  
    * ttk  
    * font
